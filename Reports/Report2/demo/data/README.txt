UA BI Demo Data (synthetic)
Este pacote contém dados sintéticos, inspirados no contexto da Universidade de Aveiro, para demonstrações em Apache Superset / PostgreSQL.

Ficheiros incluídos:
- cursos.csv
- alunos.csv
- unidades_curriculares.csv
- inscricoes_uc.csv
- orcamento.csv
- despesas.csv
- colaboradores.csv
- ausencias.csv
- projetos.csv
- publicacoes_ria.csv

Observações:
- Todos os dados são artificiais/sintéticos.
- IDs e relações foram mantidos de forma coerente para facilitar análise e dashboards.
- Os domínios cobrem académico, financeiro, RH, projetos e produção científica.
- Podem ser importados diretamente para PostgreSQL ou usados em CSV.

Sugestão de uso:
1) Criar tabelas no PostgreSQL
2) Importar os CSVs
3) Ligar o Apache Superset à base de dados
4) Criar datasets, gráficos e dashboards
